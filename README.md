 # My Personal Portfolio Website

Welcome to my first web development project! This website is a personal portfolio created as part of my coursework at **LUT University**. It showcases my life in Finland, my hobbies, and the beautiful scenery of my hometown in China.

## 📖 About This Project
The goal of this project was to build a multi-page, responsive website using **HTML5** and **CSS3**. 

### Key Features:
**Personal Profile:** A brief introduction to my background in Mechanical Engineering.
**Hobby Gallery:** A collection of photos showing my daily life and interests.
**Travel Journals:** Dedicated pages for Finland and China with optimized image galleries.
**Responsive Design:** Optimized for both desktop and mobile viewing.
**Interactive Elements:** CSS hover effects and a sticky navigation bar for better UX.

## 🛠️ How to Run Locally
If you want to run this project on your local machine:
1.  Download or clone this repository.
2.  Navigate to the project folder.
3.  Locate the **`index.html`** file.
4.  Double-click **`index.html`** to open it in your preferred web browser (Chrome, Edge, or Safari).

## 📁 Repository Structure
* `index.html`: The main landing page.
* `hobby.html`, `China.html`, `finland.html`: Sub-pages for specific content.
* `style.css`: The stylesheet containing all visual designs.
* `/images`: Folder containing all JPG and PNG assets used in the project.
